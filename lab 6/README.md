# Лабораторная работа 6: Бинарные деревья поиска

## Цель работы
Изучить древовидные структуры данных, их свойства и применение. Освоить основные операции с бинарными деревьями поиска (BST). Получить практические навыки реализации BST, рекурсивных алгоритмов обхода и анализа их эффективности. Исследовать влияние сбалансированности дерева на производительность операций.

## Теоретическая часть
Бинарное дерево поиска (BST) — это древовидная структура данных, в которой для каждого узла выполняется:
- Все значения в левом поддереве меньше значения узла
- Все значения в правом поддереве больше значения узла
- Оба поддерева также являются BST

**Основные операции и их сложность:**
- Вставка: O(log n) в среднем (сбалансированное дерево), O(n) в худшем (вырожденное дерево)
- Поиск: O(log n) в среднем, O(n) в худшем
- In-order обход: O(n) для всех случаев

**Виды обходов:**
- In-order (левый-корень-правый): возвращает значения в отсортированном порядке
- Pre-order (корень-левый-правый): полезен для копирования структуры
- Post-order (левый-правый-корень): полезен для удаления дерева

## Практическая часть

   - Сравнение производительности

3. **main.py** — демонстрационная программа
