# Лабораторная работа 1: Сравнение алгоритмов поиска

## Цель работы
Освоить понятие вычислительной сложности алгоритма. Получить практические навыки реализации и анализа линейного и бинарного поиска. Научиться экспериментально подтверждать теоретические оценки сложности O(n) и O(log n).

## Теория

### Сложность алгоритмов
- **Линейный поиск (Linear Search)**: O(n)
  - Последовательный перебор всех элементов массива
  - В худшем случае требуется проверить все n элементов

- **Бинарный поиск (Binary Search)**: O(log n)
  - Поиск в отсортированном массиве путем деления интервала поиска пополам
  - Требует предварительной сортировки массива

### Реализованные функции

#### `linear_search(arr, target)`
Линейный поиск элемента в массиве.

**Сложность**: O(n)

#### `binary_search(arr, target)`
Бинарный поиск элемента в отсортированном массиве.

**Сложность**: O(log n)

## Анализ результатов

1. **Корректность работы**: Оба алгоритма корректно находят существующие элементы (индексы 3 и 0) и возвращают -1 для отсутствующих элементов

2. **Производительность**: 
   - Бинарный поиск показывает стабильное время выполнения ~0.000006-0.000007 секунд независимо от размера массива
   - Линейный поиск демонстрирует явный рост времени с увеличением размера массива
   - На массиве из 10000 элементов бинарный поиск работает в 470 раз быстрее линейного

3. **Подтверждение теоретических оценок**:
   - Линейный поиск: O(n) - время растет пропорционально размеру массива
   - Бинарный поиск: O(log n) - время практически постоянно при увеличении n

4. **Особенности результатов**: Большое ускорение на 10000 элементах (470x) объясняется логарифмической природой бинарного поиска - при увеличении массива в 10 раз количество операций возрастает всего на 3-4

## Выводы

- Бинарный поиск значительно эффективнее линейного для больших отсортированных массивов
- Линейный поиск проще в реализации и не требует сортировки
- Выбор алгоритма зависит от размера данных и необходимости предварительной сортировки
- Теоретические оценки сложности подтверждаются экспериментальными данными
