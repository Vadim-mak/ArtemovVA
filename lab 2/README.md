Лабораторная работа №2
Основные структуры данных. Анализ и применение
Цель работы
Изучить базовые структуры данных — список, связный список, стек, очередь и дек.
Понять различия в их реализации и сложности операций, а также научиться выбирать оптимальную структуру для решения практических задач.

Теоретическая часть
В работе рассматриваются основные абстрактные типы данных:

List — динамический массив, быстрый доступ по индексу, но медленная вставка в начало.
LinkedList — связный список, быстрая вставка и удаление, но медленный доступ по индексу.
Stack — структура LIFO (последний вошёл — первый вышел).
Queue — структура FIFO (первый вошёл — первый вышел).
Deque — двусторонняя очередь с быстрыми операциями с обеих сторон.
Практическая часть
В ходе лабораторной необходимо:

Реализовать класс LinkedList с базовыми операциями вставки и удаления.
Провести сравнительный анализ производительности операций:
list vs LinkedList при вставке в начало.
list vs deque при удалении из начала.
Решить практические задачи с использованием подходящих структур данных:
Проверка сбалансированности скобок (стек).
Модель очереди печати (deque).
Проверка палиндрома (deque).
